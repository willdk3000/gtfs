Mapbox GL

Mapbox GL permet d'afficher de grandes quantités de données en utilisant le GPU plutôt que le CPU pour la visualisation de données.
Mapbox fournit aussi la librairie draw qui permet de dessiner des shapes sur la carte et de les utiliser pour des calculs. Pour voir combien de départs passe par un tronçon de rue, tracer une ligne qui croise la rue pour laquelle on veut l'information.
